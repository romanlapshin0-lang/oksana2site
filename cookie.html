Использование cookie

Настоящее уведомление применяется к посещению сайта Исполнителя.

Сайт может использовать cookie и иные аналогичные технологии для корректной работы, сохранения пользовательских настроек, повышения удобства использования сайта, аналитики и улучшения качества сервиса.

1. Какие cookie могут использоваться

  • технические — для базовой работы сайта;
  • функциональные — для запоминания пользовательских настроек;
  • аналитические — для анализа поведения посетителей и улучшения структуры сайта;
  • маркетинговые — при использовании инструментов продвижения и ретаргетинга.

2. Цели использования

  • обеспечение корректной работы страниц сайта;
  • повышение удобства использования;
  • получение обезличенной статистики и аналитики;
  • оптимизация контента, структуры и рекламных кампаний.

3. Управление cookie

Пользователь может изменить настройки использования cookie в своём браузере, ограничить или полностью запретить их сохранение. При этом часть функций сайта может работать некорректно.

4. Связь с политикой обработки персональных данных

Если информация, собираемая с помощью cookie, относится к персональным данным, её обработка осуществляется в соответствии с Политикой обработки персональных данных.
